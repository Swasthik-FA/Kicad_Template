How to fix common issues

  • lib_symbol_issues / footprint_link_issues — open the symbol/footprint library manager and add the missing library, or change the symbol/footprint to one from a configured library.
  • unconnected pins — connect the pin or place a "no-connect" flag if it is intentionally unused.
  • clearance — increase track-to-track or pad-to-track spacing, or relax the design rule for that net class if appropriate.
  • track width — match the design-rule minimum for the net class (typically 0.15 mm for signal, 0.25 mm+ for power).
  • silkscreen over pad — move the reference designator off the pad in the footprint editor or hide it.
  • schematic_parity — re-run "Update PCB from Schematic" so the layout matches the latest netlist.
  • holes too close — separate drilled holes by at least the manufacturer minimum (commonly 0.5 mm).
